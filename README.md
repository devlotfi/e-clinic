# ðŸ“‚ Project Setup

## Requirements
- Python
- Poetry
- Node JS
- Yarn

## Install

```bash
$ poetry install --no-root

# OPTIONAL: Only for tailwind compiler
$ yarn
```

## Development

```bash
$ poetry run python src/manage.py runserver

# OPTIONAL: Only for tailwind compiler
$ yarn run dev
```
